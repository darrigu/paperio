# paperio

Paper.IO like game in the terminal

## Quick Start

> [!NOTE]
> Make sure to run this program in a proper terminal

```shell
$ pip install uv         # Install project manager
$ uv sync                # Sync dependencies
$ ./server.py            # Start server
$ ./client.py            # Connect to server
```
